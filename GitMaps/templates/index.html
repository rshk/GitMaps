Welcome to GitMaps!

Advanced tools for the GitHub-powered maps.

Warning! This thing is still very experimental / proof of concept. Use at your own risk.
Usage on "real" files is not recommended as it's not guaranteed not to mess up with your files!

What's this?

GitHub recently introduced support for rendering maps from .geojson files. While I liked that feature a lot, I feel that it's a bit limited (eg. missing editing support), so I wrote an application specifically tought for dealing with maps on GitHub.

Start editing your maps now! Fork on GitHub!

Features

  1. Explore your GitHub repositories and find GeoJSON files
  2. Display your files on a map
  3. Easily edit them on a map too, thanks to the power of Leaflet Draw plugin.
  4. Save and have them automatically committed to your repository — in a way similar to what the "online edit" GitHub feature does

Technologies

  • Python
  • Flask — powerful Python micro-framework
  • Flask-RESTful — Flask extension to easily create APIs
  • rauth — for handling GitHub OAuth from Python
  • Leaflet — the map view/editing widget
  • jQuery — for communication with the API

Future plans

  • Support for displaying and editing complex multi-layer maps
    • Configuration files: something like *.gh-map
    • Map accessible as .../user/repo/branch/path.gh-map
  • Maybe, continue along the way and add support for other formats too..? We'll see...
